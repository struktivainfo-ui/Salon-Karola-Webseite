# Salon Karola – Website

Moderne, mobile-first Website für den Friseursalon **Salon Karola** in Calw-Wimberg.  
Die Website ist statisch (HTML, CSS, JavaScript) und direkt für GitHub + Vercel geeignet.

## Dateistruktur

- `index.html`
- `style.css`
- `script.js`
- `impressum.html`
- `datenschutz.html`
- `README.md`
- `assets/`
- `assets/images/`

## Lokal öffnen

`index.html` im Browser öffnen.

## Deployment (GitHub + Vercel)

1. GitHub Repository erstellen.
2. Dateien hochladen.
3. Vercel mit GitHub verbinden.
4. Framework Preset: `Other`.
5. `Deploy` klicken.

## Bilder einfügen

Bilder gehören in `assets/images/`.

Empfohlene Dateinamen:

- `salon-aussen.jpg`
- `salon-innen.jpg`
- `team-jessi.jpg`
- `team-ute.jpg`
- `styling-1.jpg`
- `styling-2.jpg`
- `styling-3.jpg`
- `beratung.jpg`
- `lotto.jpg`

Empfohlene Bildgrößen:

- `1600px` Breite für große Bilder (Hero, Willkommen)
- `800px` Breite für Team und Galerie

Bilder vor dem Upload komprimieren, damit die Seite schnell bleibt.

## Logo

Logo-Datei: `assets/images/logo-salon-karola.jpg` oder `assets/images/logo-salon-karola.png`.

Wenn keine Logo-Datei vorhanden ist, zeigt die Website automatisch ein Textfallback an.
